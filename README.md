# sword-cloud
    将常用的技术栈整合起来，包括 Dubbo RPC 、HTTP RPC协议，平台网关、Oauth2.0、搜索引擎、分布式事务、限流、熔断、降级、链路追踪，可用于快速开发SpringCloud 应用服务

``` lua
sword-cloud
├── sword-common -- 工具类及通用代码
├── sword-gateway -- 网关
├── sword-oauth -- 认证授权
├── sword-services -- 平台服务聚合
├── sword-starters -- 平台自有starters聚合
└── sword-ops -- 运维管理模块
```
